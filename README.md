# ElectroWallet for WebProgramming
##### Frontend:
- HTML/CSS
- Bootstrap Material
- Javascript (jQuery)
##### Backend:
- PHP (Symfony), MySql (Doctrine)
##### Requirements:
- PHP v7.1.3
### How to install:
```sh
$ composer install
$ php bin/console doctrine:migrations:migrate
$ npm install
$ yarn encore dev
```
### Woala, you are ready to use this app
